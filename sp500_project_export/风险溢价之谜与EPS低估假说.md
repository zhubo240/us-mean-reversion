# 风险溢价之谜与 EPS 低估假说

## 讨论背景

在分析 S&P 500 均值回归项目过程中，从股息机制出发，逐步推导出一个关于风险溢价之谜的解释假说，并用 Shiller 历史数据进行了初步验证。

---

## 一、股息的本质

### 派息不影响 EPS

EPS（每股收益）是利润表的结果，在派息决策之前已确定。派息从公司留存收益/现金中支出，不影响利润表。

```
EPS 和派息是两个独立环节：
  EPS = 净利润 / 总股数（利润表结果）
  派息 = 从现金/留存收益中支出（资产负债表变动）
```

### 股价的机械性调整

除息日（ex-dividend date），交易所自动将股价下调约等于每股股息的金额：

```
派息前：公司 = 业务价值 $98 + 现金 $2 = $100/股
派息后：公司 = 业务价值 $98 + 现金 $0 = $98/股
股东手里：$98 股票 + $2 现金 = 还是 $100
```

**派息本质上是公司把自身价值的一部分以现金形式送给股东。** 股价下降的比例就是股息收益率（dividend yield）。S&P 500 历史股息收益率约 2-4%。

### 股价为什么会涨回来

因为公司**持续赚钱**。派出去的只是利润的一部分（派息率约 30-50%），赚钱能力没变，新的利润会填上缺口。PE 能否恢复，取决于市场是否相信公司盈利能力不变。

### 派多派少不影响总回报

```
大方公司：派 $4，留 $1 再投资 → 股息高，EPS 增长慢
抠门公司：派 $1，留 $4 再投资 → 股息低，EPS 增长快
总回报 ≈ 差不多，只是形式不同
```

这就是为什么 Berkshire Hathaway 不派息 — 留下利润再投资，通过股价上涨回馈股东。

### 关键推论

S&P 500 名义 EPS 年化增长 ~5.4%，是在派息 ~3% 的前提下的结果。如果不派息，EPS 可能增长更快。**~10% 的名义回报中，扣除 PE 扩张约 1-1.5%，剩余 ~8.5% 全部来自企业盈利能力**，股息和 EPS 增长只是同一盈利能力的不同表达形式。

## 二、回报的 DCF 解释

### 为什么股价会上涨

股价已经 price in 了未来盈利预期。股价上涨不是因为"新信息"，而是**折现率的兑现**：

```
今天：把未来所有盈利按 10% 折现回来 → 股价 $100
一年后：同样的未来盈利近了一年 → 股价 $110
```

就像折价买的债券，价格每天都在往面值靠拢。S&P 500 的 ~10% 年化回报，本质上就是市场要求的风险补偿（折现率）。

### 折现率的构成

```
折现率 ≈ 无风险利率 + 风险溢价
  ~10%  ≈   ~4-5%    +   ~5-6%
```

- **无风险利率**：国债收益率，纯粹的时间价值补偿
- **风险溢价**：股票比国债多出的回报，补偿不确定性

### 风险溢价是什么

投资者承担不确定性所要求的额外回报。国债到期一定还钱，股票可能亏损、破产。这个额外的 5-6% 把投资者"吸引"过来承担风险。

### Alpha 和 Beta

```
个股回报 = α + β × 市场回报

β（beta）= 跟着市场走的部分（无风险利率 + 风险溢价），买指数就能拿
α（alpha）= 定价误差修正 / 选股能力，零和游戏
```

---

## 三、风险溢价之谜

### 问题

风险溢价不是来自"投资者内心的理性模型"。因果关系实际上是反的：

```
不是：风险 → 市场给了溢价
而是：观察到溢价 → 模型需要参数解释 → 叫它"风险补偿"
```

DCF 框架发现用无风险利率折现对不上实际股价，于是加了一个"风险溢价"参数。这个参数本质上是个 **residual（残差）**，打包了所有东西：真实风险补偿、非理性恐惧、流动性偏好、行为偏差等。

### Mehra-Prescott 之谜

按照理性模型，人类正常的风险厌恶程度只能解释 ~1-2% 的溢价。实际观察到 5-6%，高得离谱。市场表现得比理论上应该的**怕风险得多**。

---

## 四、EPS 低估假说

### 核心逻辑

```
市场基于预期 E 定价 → 实际 E 增长超预期 → PE 被压缩
→ 市场修正 P 向上恢复 PE → 这个 P 上涨 = 额外收益
→ DCF 假设定价合理 → 额外收益被归入"风险溢价"
```

**假说：风险溢价的一部分，是市场长期低估盈利增速的系统性误差。**

### 为什么低估可以持续

- 人类天生不擅长理解**复合增长**
- 每一代人都经历过衰退和危机，对未来本能偏悲观
- 技术进步、生产率提升这些长期驱动力难以预测

### 如果假说成立

```
风险溢价 ≈ 真实风险补偿 + 盈利增速低估补偿
  5-6%   ≈    ？%        +      ？%
```

## 五、数据验证

### 方法

1. **回报分解**：将 S&P 500 历史名义总回报拆解为 股息收益率 + EPS 增长 + PE 变化
2. **Gordon 模型检验**：用 `g_implied = r - D/P`（r = 国债利率 + 2% 合理风险溢价）反推市场隐含增长预期，对比实际 EPS 增长
3. **数据来源**：Robert Shiller 历史数据（S&P 500 价格、EPS、股息、CPI、国债利率）

### 验证代码

- `code/verify_eps_underestimation.py` — 假说验证主脚本
- `code/corrected_returns.py` — 修正后的回报计算

### 回报分解结果（1928-2024，96 年）

| 分解项 | 年化贡献 |
|--------|---------|
| 股息收益率 | 2.90% |
| EPS 增长 | 5.16% |
| PE 扩张 | 0.81% |
| **名义总回报** | **8.91%** |

### Gordon 模型检验结果

在"合理风险溢价 2%"假设下，比较市场隐含增长预期 vs 实际 EPS 增长：

- **低估次数：13/20 个时段（65%）**
- **平均低估幅度：1.66%/年**

按时代分布：
- 低利率时代（1935-1975）：几乎全部低估，市场严重低估了经济起飞期的盈利增长
- 高利率时代（1980-2000）：反而高估，利率高导致隐含预期被拉高
- 2010 年后：又开始低估，低利率 + 科技公司盈利爆发

## 六、修正后的回报归因

### 回报归因总结

```
名义总回报 8.91% =

  无风险利率（时间价值）       4.88%
  + 合理风险补偿               2.00%  ← 理性承担风险的报酬
  + EPS 低估修正               1.66%  ← 本假说
  + 残余（恐惧/偏差/幸存者）   0.37%
  ─────────────────────────
  合计                         8.91%
```

### 关键发现

1. **EPS 低估假说解释了 1.66%**，加入后风险溢价之谜基本消失（残余仅 0.37%）
2. **修正后的风险溢价为 2.37%**，接近 Mehra-Prescott 的合理水平（~2%）
3. PE 从 11.1 扩张到 24.0，年化仅 0.81%，有长期向上修正但幅度有限

### 96 年复利影响

| 场景 | 年化回报 | $100 增长到 |
|------|---------|------------|
| 实际回报 | 8.91% | $361,804 |
| 修正后回报（去除低估红利） | 7.25% | $82,960 |
| 纯国债 | 4.88% | $9,682 |

**EPS 低估给投资者额外带来 $278,844，占总财富的 77%。**

## 七、结论

### 假说状态

EPS 低估假说得到**初步支持但非决定性证据**：

- 65% 的时段实际增长超过隐含预期（有倾向性，但不是压倒性）
- 低估幅度（1.66%）能很好地弥合风险溢价之谜的缺口
- 时代特征明显：经济扩张期低估更严重，高利率期反而高估

### 方法论局限

1. **循环论证风险**：用市场价格推预期，又用市场结果验证
2. **假设敏感性**："合理风险溢价 2%"是外部假设，改变这个数字会大幅影响结论
3. **数据精度**：使用约每 5 年的采样点，非连续年度数据

### 更干净的验证路径

| 方法 | 数据需求 | 是否需要公司级数据 |
|------|---------|-----------------|
| 指数级 Shiller 数据（已完成） | 公开免费 | 否 |
| I/B/E/S 分析师预测 vs 实际（最干净） | 付费，1985年起 | 否 |
| 个股 EPS 预测偏差分布 | I/B/E/S + CRSP | 是 |

**指数级别验证不需要公司级别数据。** 公司级别数据的价值在于另一个问题：哪些行业/公司的 EPS 被低估得最严重（选股层面的 alpha 来源分析）。

### 更广泛的启示

风险溢价可能不是单一原因，而是多因素叠加：

```
观察到的风险溢价 ~4-5% ≈
  真实风险补偿    ~2%     （理性的，Mehra-Prescott 水平）
  + 盈利低估修正  ~1.7%   （系统性认知偏差）
  + 行为偏差残余  ~0.4%   （恐惧、短视、幸存者偏差等）
```

这个分解暗示：**长期投资者的超额回报，很大程度上来自于人类对复合增长的认知缺陷和对风险的过度恐惧。**
